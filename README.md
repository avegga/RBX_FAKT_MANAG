# RBX_FAKT_MANAG

Локальное MVP-приложение для загрузки табличных фактов, настройки типов и отображения, фильтрации, парсинга строк, просмотра журнала событий и сохранения результата в XLSX.

## Запуск

1. Создать виртуальное окружение:

```powershell
python -m venv .venv
```

2. Активировать окружение в PowerShell:

```powershell
.\.venv\Scripts\Activate.ps1
```

Если PowerShell блокирует запуск скриптов, используйте:

```powershell
Set-ExecutionPolicy -Scope Process Bypass
```

3. Установить зависимости:

```powershell
pip install -r requirements.txt
```

4. Запустить приложение:

```powershell
python app.py
```

5. Открыть в браузере адрес `http://127.0.0.1:5000`.

## Что настроить перед работой

Во вкладке Настройки укажите:

- папку результата;
- путь к текстовому файлу инструкций.

## Основные сценарии MVP

- загрузка CSV или XLSX в текущую сессию;
- применение типов из шаблона и пересчёт ошибок;
- фильтрация, сортировка, управление видимостью и порядком столбцов;
- парсинг по выбранному столбцу с одним шагом отката;
- просмотр инструкций из внешнего текстового файла;
- просмотр событий текущей сессии и недавней истории;
- сохранение результата в XLSX по текущему виду таблицы.
